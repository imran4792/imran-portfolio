# Pattan Mohammad Imran Khan Portfolio

A futuristic developer portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion. The portfolio highlights Java Full Stack and DevOps experience with a dark glassmorphism UI, animated interactions, and responsive layout.

## Features
- Hero section with animated role typing and CTA anchors
- About section with professional summary and backend-first mindset
- Skills section with animated progress bars
- Featured project card for a Flight Booking Application
- Experience timeline and blog preview cards
- Resume download section and contact form
- Responsive, premium glassmorphism design

## Local Development
1. Install dependencies
   ```bash
   npm install
   ```
2. Start development server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```
4. Export static site for GitHub Pages
   ```bash
   npm run export
   ```

## Deployment
This project is configured for static export with `next export` and can be hosted on GitHub Pages or any static hosting provider.
